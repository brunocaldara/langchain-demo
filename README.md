# langchain-demo

## Instructions for run project:

### Create virtual env:

- python3 -m venv .venv

### Activate virtual env:

- source .venv/bin/activate (linux/mac)
- .\env\Scripts\activate.bat (windows)

### Install dependencies:

- pip install -r requirements.txt
